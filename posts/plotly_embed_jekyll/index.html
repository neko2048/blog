Overview

Preparation

We would need some packages to help us:

1
2
3

plotly #-> Draw interactive figures
numpy  #-> Load Data as an array
PIL    #-> Load figures we alreadly have

Install Plotly

Create an virtual environment is recommended first.

Plotly can be install using pip:

1

$ pip install plotly

or using conda:

1

$ conda install -c plotly

HTML file to _include folder

And we can first open html file in browser to check how it is like. After that, copy it as {GitPage_Repository}/_include/self_define/plotly_fig.html.

The self_define folder is created for what we special defined.

Display on Gitpage

Then we can just link to the gitpage we want to show the result:

1

{ % include self_define/plotly_fig.html % }
